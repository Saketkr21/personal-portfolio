GitStats

A small app that uses the GitHub API to filter the most stars and forked repositories with the option to filter by language. The main purpose of this project for me personally was to get an idea of the benefits of using PreactJS over React so I could then apply it at Sky. It was a good learning process in which I had to do a lot more reading into the framework that I normally would do to learn how PreactJS can offer the same functionality as React but at a smaller bundle size. I also opened this project to Hacktoberfest which gave me experience opening up a project to become more open sourced.
